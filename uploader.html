ℹ️ Что происходит

  
    
    
  • Обложки генерятся отдельно в covers/ как и раньше — этот загрузчик их не трогает, только JSON.
    • 
    
  • Для remote-режима обложки будут как буквы-инициалы, пока владелец не скопирует файлы из library/files/ в covers/ и не запустит sync_covers.py.
    • 
    
  • Сайт сам делает fetch(gamesUrl) + fetch(overridesUrl) и мерджит с основным каталогом (старые игры из каталога никогда не удаляются).
    • 
    
  • По умолчанию заливка игр — аддитивная: сначала читается текущая корзина и к ней добавляются только новые названия, уже лежащие там игры не пропадут даже при неполном library.json. Чекбокс «Полностью заменить список» делает зеркальную замену (для чистки).
    • 
    
  • Рекомендуемое хранилище — Pantry (getpantry.cloud): запись из браузера одним POST, ключей не надо. Gist — только руками на сайте gist.
